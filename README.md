# Maternal Health Monitoring Dashboard

A web-based application designed to monitor maternal health vitals and assist in early risk detection using rule-based classification.

## Features
- Manual input of health vitals (Glucose, Hemoglobin, Cortisol, Heart Rate, Blood Pressure)
- Rule-based classification to identify normal and abnormal conditions
- Health data tracking and history management
- Trend visualization of patient data
- CSV export functionality

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Project Overview
This system simulates a maternal health monitoring solution where users can input vital parameters and receive instant feedback based on predefined medical thresholds. It aims to support early identification of potential health risks in pregnant women.

## Key Highlights
- Built an interactive dashboard for real-time data monitoring
- Implemented condition classification logic based on input values
- Designed user-friendly interface for efficient data handling
